# 330 → 340 견적서 전환

> 작업지시서(330) → 견적서(340) 산출 규칙. **사용자와 하나씩 확인해 확정한 내용.**
> 아직 연결되지 않은 부분은 **n/a로 표시**하고, 구현 시 [§3 연결 대기 목록](#3-연결-대기-목록-미연동--현재-na)을 따라 연결한다.

## 1. 견적 행 생성 단위
- **인쇄정보 1줄**(예 `본책-표지1`, `본책-본문1`)당 → **CTP · 용지 · 인쇄 · (후가공)** 세트 생성. **대수 무관.**
  - **CTP**: 제작에서 **CTP 체크** 시
  - **인쇄**: 제작에서 **인쇄 체크** 시
  - **용지**: 항상 생성 (선방이면 단가류 숨김)
  - **후가공**: 인쇄행에 후가공 옵션(`data-postprocess`)이 있으면
- **제본**: 제본정보 줄별로 생성
- **포장 / 배송 / 기타**: 공통 1줄씩

## 2. 행별 산출 규칙 (확정)

### CTP
- 색도 = 도수A + 도수B *(수량에 포함. 표시는 컬러/흑백 구분용)*
- 수량 = **대수 × 색도** (대수 = 324 배열계산 결과)
- 단가 = 단가표 적용 등급의 **CTP출력** (A등급 = 5,000)
- 금액 = 수량 × 단가
- 규격 = **13_1_5_2 인쇄장비**(13_1_5_1은 CTP장비로 제판종류 없음) 중 그 그룹(R5/R8)의 **첫 장비 제판종류** — 참고용, 계산 무관

### 용지
- 수량(R) = 작업지시 그 줄의 **모든 대수 합계R 총합** = Σ(정미+손지)/500
- 고시가 = 8_5_0 가격정보 고시가 / 할인 = 8_5_0 기준할인율(사용자 수정 가능) / 단가 = 고시가 × (1−할인/100) / 금액 = 수량 × 단가
  - **연결 방식**: 320 용지 선택 팝업이 8_5_0 등록 용지(고시가·기준할인율·인쇄단가적용)를 직접 보유 → 선택 시 행 `data-paper-listprice/discount/printapply`에 저장 → 320·340으로 전달. 표준원가는 별도(원가 마스터 미연동 → n/a)
- **선방** = 단가·견적가·할인·고시가 **숨김** (용지명·규격·수량·공급방식만)
- 손지 = **330에서 수정한 값 반영**

### 인쇄

- 사이즈 = 용지의 **8_5_0 인쇄단가적용 = 전지규격명(specName)**: 국전/소국전/4*6전/대국전 *(용지 등록 시 지정한 전지규격이 곧 인쇄단가적용)*. 변규격은 개별 지정
- 도수컬럼 = **max(도수A,도수B) ≥ 3 → 4도**, 둘 다 ≤ 2 → 2도/1도 (2도·1도 = 1/1·2/1·2/2·1/2)

- 금액 = **수량(R) × 단가 × 색도** (본문·표지 공통)

### 후가공 — 박작업 (325 비용계산, 13_1_8 박작업 단가표 기준)
> **진행상태(2026-06-11)**: 325 박작업 카드 **곳마다 면적 개별(다중 행) UI + Σ 합산 산식** 구현 완료
- **입력(325 박작업 카드)**: 박종류(금박·은박·안료박·홀로그램박·형압) · 양단면 · 기존 동판 보유(체크) · 동판종류(아연·구리·형압용) · **박 면적 다중 행**(곳마다 가로×세로 mm, +면적 추가/삭제, 초기 1행·마지막 1행 유지)
- **mm→cm 변환**(÷10). 면적cm² = (가로mm/10)×(세로mm/10)
- **가공비** *(곳별 합산)*: 곳별가공비 = `부당단가(=max(5*8단가, 1cm²단가×면적cm²)) × 부수`. **부수 ≤ 기본비적용수량(1,000)** 이면 곳별로 **기본비 하한** 적용 → `Σ 곳별 max(기본비, 곳별가공비)` / **초과** 면 하한 없음 → `Σ 곳별 곳별가공비`
  - *기본비 = 단가 계산값이 기본비보다 낮을 때 받는 하한. 적용수량 초과 부수에서는 기본비보다 낮아도 하한 미적용*
- **동판비**(미보유 시) *(곳별 합산)*: **Σ(곳별 max(동판기본비, 동판단가 × (가로cm+2)(세로cm+2)))** *(사방 1cm 여유. 동판기본비도 단가가 그보다 낮으면 받는 하한, 수량 조건 없음)*
- **총 = (가공비 + 동판비) × (양면 ? 2 : 1)**
- **단가**: 13_1_8 A등급 — 325 내장 `FOIL_PRICE`(금/은/안료 cm²1.5·5\*8 60·기본비70,000 / 홀로그램 3.75·150 / 형압 1.25·50·60,000) · `PLATE_PRICE`(아연 160/11,200·구리 200/14,000·형압용 250/17,500)
- **곳수(N) 집계**: 가로·세로 **둘 다 입력된 유효 행만** 곳수·합산에 포함(빈 행 무시). 유효 행 0개면 0원
- **구현**: 면적 행 `.foil-area-row`(`.foil-w`/`.foil-h`) 다중 → `foilAreaRowHTML`/`addFoilArea`/`removeFoilArea`. `calcFoilCost(card, qty, isDouble)`가 유효 행 배열로 Σ 합산. `calculateCost()`에서 `.foil-areas` 보유 시 호출 → 공급가. 부수 = `currentPrintInfo.qty`(320 orderQty)
